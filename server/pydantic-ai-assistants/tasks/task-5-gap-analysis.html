Task 5

Gap Analysis Between Resume & Job Description

Learning Objectives

  • Use LLM to analyze resume vs. job description
  • Structure analysis output with Zod schemas
  • Display results in organized sections
  • Understand prompt engineering for analysis

The Gap Analysis Schema

Before calling the LLM, we define the exact structure we want back. This is the Zod schema used by our backend:

Backend Schema (Zod)
const GapAnalysisSchema = z.object({
  key_requirements: z.array(z.string()),
  relevant_experience: z.array(z.string()),
  gaps: z.array(z.string()),
  potential_strengths: z.array(z.string())
});

This schema ensures the LLM returns:

  • key_requirements: Main skills/qualifications from the job description
  • relevant_experience: Matching skills and experience from the resume
  • gaps: Missing or weak areas in the resume
  • potential_strengths: Valuable skills not in job description but could strengthen application

How It Works

  1. User Input: Resume and job description text
  2. API Call: POST to /api/structured/resume-analysis
  3. Backend: Converts Zod schema to JSON Schema
  4. OpenAI: GPT-4 generates structured analysis matching the schema
  5. Validation: Zod validates the response before returning
  6. Display: Results rendered in 4 organized sections
Behind the Scenes: The backend uses response_format to guarantee OpenAI returns valid JSON matching our Zod schema. This prevents malformed outputs that would crash the application.

The Prompt

Here's the prompt template sent to GPT-4 (simplified):

Analyze this resume against the job description.

Resume:
${resume_text}

Job Description:
${job_description_text}

Return JSON with these exact fields:
- key_requirements: array of strings (main skills from job)
- relevant_experience: array of strings (matching skills from resume)
- gaps: array of strings (missing areas in resume)
- potential_strengths: array of strings (valuable skills not in job description)

Practice Opportunity

Try these experiments:

  • Replace the job description with a real posting from LinkedIn
  • Update the resume with your own experience
  • Notice how the gaps change based on the job requirements
  • Observe which strengths are identified that aren't in the job description

Expected Output Structure

After clicking "Analyze Resume," you'll see 4 color-coded sections:

  • 🎯 Key Requirements (blue) - What the job is looking for
  • ✅ Relevant Experience (blue) - What the resume demonstrates
  • ⚠️ Gaps to Address (yellow/warning) - What's missing or weak
  • 💪 Potential Strengths (green/success) - Bonus skills that could help

This structured analysis gives job seekers actionable insights on how to improve their resume for the specific role.
