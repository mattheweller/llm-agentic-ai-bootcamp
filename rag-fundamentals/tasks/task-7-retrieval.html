How Semantic Search Works

To find relevant chunks, we:

  1. Embed the query - convert user question to a vector
  2. Calculate similarity - compare query vector to all chunk vectors
  3. Rank by similarity - highest scores = most relevant
  4. Return top-k - retrieve the k most similar chunks

💡 Cosine Similarity: Measures the angle between two vectors. Score of 1.0 = identical, 0.0 = orthogonal, -1.0 = opposite.

Code Example: Cosine Similarity

// Calculate cosine similarity between two vectors
function cosineSimilarity(vecA, vecB) {
  let dotProduct = 0;
  let magnitudeA = 0;
  let magnitudeB = 0;

  for (let i = 0; i < vecA.length; i++) {
    dotProduct += vecA[i] * vecB[i];
    magnitudeA += vecA[i] * vecA[i];
    magnitudeB += vecB[i] * vecB[i];
  }

  return dotProduct / (Math.sqrt(magnitudeA) * Math.sqrt(magnitudeB));
}

// Semantic search
async function search(query, topK = 3) {
  // 1. Embed the query
  const queryEmbedding = await apiClient.embed(query, 'openai');

  // 2. Calculate similarity with all chunks
  const results = vectorStore.map(item => ({
    ...item,
    similarity: cosineSimilarity(queryEmbedding, item.embedding)
  }));

  // 3. Sort by similarity (descending)
  results.sort((a, b) => b.similarity - a.similarity);

  // 4. Return top-k
  return results.slice(0, topK);
}
