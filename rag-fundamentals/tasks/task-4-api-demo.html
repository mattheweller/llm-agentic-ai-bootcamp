Try asking a question. The request will be securely proxied through our backend (your API key never reaches the browser):

Response will appear here...

Code Example

Here's how the API client works behind the scenes:

// Initialize the API client
const client = new LLMAPIClient('http://localhost:3000');

// Simple chat request
const response = await client.simpleChat(
  'Explain RAG in one sentence',
  'openai',  // or 'anthropic' or 'google'
  { temperature: 0.7 }
);

console.log(response);  // AI's response text

Try It: Generate Embeddings

Embeddings convert text into numerical vectors for semantic search. Try generating an embedding:

Embedding vector will appear here...
// Generate embeddings
const embedding = await client.embed(
  'Retrieval Augmented Generation',
  'openai'
);

console.log(embedding);  // Array of 1536 numbers
console.log(embedding.length);  // 1536

Key Takeaways

  • Security First: API keys are stored server-side in .env, never in the browser
  • Proxy Pattern: Frontend calls localhost:3000/api/*, backend calls LLM providers
  • Unified Interface: Same API client works with OpenAI, Anthropic, and Google
  • Error Handling: Built-in timeout and error handling for robustness
