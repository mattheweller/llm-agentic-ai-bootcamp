System Architecture

Your complete RAG system flow:

User Question
      ↓
[1] Embed Query (OpenAI API)
      ↓
[2] Semantic Search (Cosine Similarity)
      ↓
[3] Retrieve Top-K Chunks
      ↓
[4] Build Prompt (Context + Question)
      ↓
[5] Generate Answer (OpenAI API)
      ↓
Display to User
