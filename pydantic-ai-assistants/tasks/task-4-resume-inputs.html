Resume AI Assistant Inputs

Before we can analyze or generate resumes, we need two key pieces of information:

  1. Current Resume: The user's existing resume text
  2. Target Job Description: The job posting they're applying to

This task sets up input forms with sample data you can use or replace with your own.

Data Persistence

When you click "Save Inputs," your data is stored in browser localStorage. This means:

  • ✅ Data persists across page refreshes
  • ✅ Inputs carry forward to gap analysis and resume generation tasks
  • ✅ No server-side storage - your data stays private in your browser
  • ⚠️ Clearing browser data will delete stored inputs
LocalStorage Implementation
// Save to localStorage
function saveInputs() {
  const resume = document.getElementById('resumeInput').value;
  const jobDesc = document.getElementById('jobDescInput').value;

  localStorage.setItem('pydantic_resume', resume);
  localStorage.setItem('pydantic_jobdesc', jobDesc);

  showStatus('✅ Saved successfully!', 'success');
}

// Load from localStorage on page load
function loadSavedInputs() {
  const resume = localStorage.getItem('pydantic_resume');
  const jobDesc = localStorage.getItem('pydantic_jobdesc');

  if (resume) document.getElementById('resumeInput').value = resume;
  if (jobDesc) document.getElementById('jobDescInput').value = jobDesc;
}
