Module 07

Agents - N8N

Automate business processes with no-code AI workflows integrating Gmail, Google Sheets, and Calendar APIs.

About This Module

Learn to build production AI workflows using N8N, a powerful no-code automation platform. This module covers progressive workflow complexity from basic LLM calls to multi-step pipelines with memory, search, and Google service integrations.

N8N Setup Required

This module requires an N8N instance (cloud or self-hosted) and Google OAuth credentials for Gmail/Sheets/Calendar integration. Budget 30-60 minutes for initial OAuth setup following the N8N documentation.

What You'll Learn

  • N8N workflow fundamentals and node architecture
  • LLM integration with memory and context management
  • External API integration (Tavily search, Google services)
  • Structured output validation with JSON schemas
  • Email triggers and classification workflows
  • Multi-step business process automation

Learning Resources

🔄 8 Workflow Files

Progressive N8N Workflows

8 JSON workflows progressing from basic summarization to complete multi-step automation pipelines.

01 - Basic Summarization 02 - Memory + Search 03 - Google Sheets 04 - Structured Output 05 - Calendar Tools 06 - Email Trigger 07 - Complete Pipeline Practice Solution
📋 Workflow Pattern

Progressive Complexity

Start simple with basic LLM calls, progressively add memory, search, external integrations, and triggers.

Pattern: Basic → Memory → Search → Integrations → Triggers → Complete

Prerequisites

  • N8N account (cloud.n8n.io or self-hosted instance)
  • OpenAI API key (required)
  • Gmail account with OAuth credentials
  • Google Sheets API access
  • Google Calendar API access (for workflows 5-7)

Key Technologies

N8N OpenAI Gmail API Google Sheets Calendar API

Getting Started

  1. Set up N8N instance (cloud.n8n.io recommended for beginners)
  2. Configure Google OAuth credentials following N8N Google OAuth docs
  3. Add OpenAI API credentials in N8N settings
  4. Download workflow 01 and import into N8N (Workflows → Import from File)
  5. Test the basic workflow, then progressively import and test workflows 02-07
  6. Study the practice solution workflow to reinforce learning

Important Notes

  • Workflows are JSON files - they must be imported into N8N, not executed as scripts
  • Google OAuth setup is the most time-consuming step - complete this first
  • Test each workflow incrementally before moving to the next
  • Workflow 06-07 use Gmail triggers - ensure proper permissions are granted
