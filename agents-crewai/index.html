Module 06

Agents - CrewAI

Build role-based agent teams for data science automation with custom tools and task delegation.

About This Module

Learn to build production-ready agent teams using CrewAI, a framework for orchestrating role-based AI agents. This module covers custom tool development, agent coordination, and applying AI agents to data science workflows including ML model training.

What You'll Learn

  • CrewAI agent architecture and role design
  • Building custom tools with Pydantic validation
  • Agent team coordination and task delegation
  • Data science automation with pandas and scikit-learn
  • Predictive analytics with XGBoost and ML pipelines
  • Production-ready tool patterns (notebookExecutor)

Prerequisites

  • Python 3.8 or higher
  • OpenAI API key (required)
  • Basic understanding of pandas DataFrames
  • Familiarity with data science workflows (helpful)

Key Technologies

CrewAI pandas scikit-learn XGBoost Custom Tools

Getting Started

  1. Download both notebooks and the custom tool using the buttons above
  2. Place notebookExecutor.py in the tools/ directory
  3. Download the sample CSV to the data/ directory
  4. Open Notebook 1 (Data Science Automation) in Jupyter or Colab
  5. Configure your OpenAI API key in a .env file
  6. Install dependencies: pip install crewai pandas numpy scikit-learn xgboost
  7. Complete Notebook 1, then proceed to Notebook 2 (Predictive Analytics)

Custom Tool Pattern

The notebookExecutor.py tool demonstrates production patterns for CrewAI custom tools including Pydantic validation, namespace management, and automatic dependency installation. Study this tool to learn how to build robust agent capabilities.
